CSS Grid

CSS Grid is a powerful layout system now available in most browsers as of January 2017.

This is something to get excited about. Previously developers used frameworks such as Bootstrap and Skeleton to create grid layouts in CSS. Grid allows developers to separate their HTML from CSS layout and drop those heavy, high maintenance frameworks.

CSS Grid introduces properties that allow us to create grid structures and control the placement and sizing of grid items. This means that we can use media queries to adapt our grids to different layouts. Grid also comes with a powerful auto-placement system, which makes it easier to fill up available space without having to do calculations.

For more information on CSS grid or help transitioning from Bootstrap to CSS grid, I recommend reading Zells blog on the subject.

https://zellwk.com/blog/from-html-grids-to-css-grids/
